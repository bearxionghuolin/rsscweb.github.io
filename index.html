Abstract

Compared with conventional visual transmission, remote sensing image transmission faces critical challenges such as long satellite downlink latency and interrupted UAV aerial communication in post-disaster scenarios. To address these issues, this paper proposes a novel semantic communication method for communication-constrained remote sensing, named RSSC (Remote Sensing Semantic Communication). RSSC adopts a two-stage framework. In Stage One, a sLoRA strategy is designed to lightly fine-tune the multimodal large language model for accurate key target detection, yielding bounding boxes and textual descriptions; it further generates instance-aware global descriptions using zero-shot capability. In Stage Two, a diffusion model is adopted to reconstruct remote sensing images from the transmitted text and target positions. By encoding images into compact textual data, RSSC reduces transmission volume to the byte level with extremely low overhead.

Extensive experiments on RSOD and RescueNet datasets validate the effectiveness of RSSC. On RSOD, Stage One achieves an F1-score of 0.9655, and the reconstruction results are 42.87 FID and 0.1482 SSIM. On RescueNet, the F1-score is 0.8022, with 40.79 FID and 0.1513 SSIM. Based on a systematic review of existing works, RSSC achieves the highest information compression ratio among remote sensing image semantic communication methods, and delivers state-of-the-art reconstruction quality among layout-guided remote sensing image generation approaches.

Key Contributions

Byte-Level Semantic Compression

RSSC encodes remote sensing images into compact textual data (descriptions + bounding boxes), achieving the highest compression ratio among remote sensing semantic communication methods. Only text is transmitted—no visual features or thumbnails.

sLoRA Fine-Tuning Strategy

A selective LoRA approach that fine-tunes only 0.1% of MLLM parameters (vision encoder layers 24-31, all connector layers, and top LLM layers), enabling efficient adaptation of Qwen2.5-VL-7B to remote sensing with minimal training cost.

BTFusion Module

A bounding box-text fusion mechanism that encodes spatial information via Fourier embedding and fuses it with Stable Diffusion's visual features through self-attention, enabling reconstruction from text and box coordinates alone.

Zero-Shot Global Description

Leverages MLLM's zero-shot capability to generate instance-aware global descriptions without additional training, balancing key target information with scene context for comprehensive semantic representation.

Method

RSSC is a two-stage semantic communication framework. Stage One handles encoding at the transmission end; Stage Two handles reconstruction at the receiving end.

Stage One · Encoder (Transmission End)

MLLM-based Semantic Encoding

A multimodal large language model (Qwen2.5-VL-7B) is fine-tuned via the proposed sLoRA strategy to master remote sensing target detection. For each input image:

  • Detect key targets → output bounding boxes + category labels
  • Construct structured instance prompts
  • Generate a global, instance-level textual description via zero-shot prompting
  • Output: compact text (~byte level) ready for transmission
Stage Two · Decoder (Receiving End)

Diffusion-based Image Reconstruction

A pre-trained Stable Diffusion model with the novel BTFusion module reconstructs the remote sensing image from the received text and bounding box information:

  • BTFusion encodes bounding box positions into spatial feature tokens
  • Bounding box features are concatenated with SD's visual features
  • Self-attention mechanism fuses positional and visual context
  • Output: high-quality remote sensing image reconstruction

The sLoRA fine-tuning strategy — selectively applying LoRA to visual encoder, connector, and language layers for lightweight MLLM adaptation.

The BTFusion module — encoding bounding box features and concatenating them with Stable Diffusion's visual features for spatially faithful reconstruction.

Conclusion

This work designs an efficient semantic communication method for remote sensing visual information — RSSC. Stage One of RSSC encodes remote sensing visual information into descriptive text and bounding box text, while Stage Two reconstructs remote sensing visual information with high quality based on such information, compressing the volume of transmitted information to the byte level, which has extensive application value in remote sensing scenarios with communication difficulties (satellite downlink, post-disaster UAV, etc.).

In terms of quantitative metrics, RSSC achieves state-of-the-art reconstruction quality among layout-guided remote sensing image generation methods, while achieving the highest information compression ratio among remote sensing image semantic communication methods. The GPU memory requirement (~9.2 GB after INT8 quantization) is well within the capacity of modern edge computing platforms deployed on satellites and professional UAVs, demonstrating complete actual deployment capability and application value.
